# PRD: Myblog 个人主页 — 第一周搭建

## 目标

搭建一个简洁、有美感的个人博客主页，用于发布日常感想和学习记录。第一周产出：可本地运行、可部署的 MVP。

## 技术决策

| 项 | 选择 | 理由 |
|---|------|------|
| 框架 | Astro | 静态优先，Markdown 原生支持，零 JS 输出 |
| 样式 | Tailwind CSS | 原子化 CSS，快速实现简洁设计 |
| 内容 | Markdown 文件 | 本地编辑，git 版本控制，无需 CMS |
| 部署 | GitHub Pages | 免费，push 即发布 |
| 后端 | 无 | 纯静态站，无需数据库 |

## 第一周范围（MVP）

### 页面

1. **首页 `/`** — 文章列表，每篇显示标题、日期、摘要
2. **文章页 `/posts/[slug]`** — 单篇文章详情，Markdown 渲染
3. **关于页 `/about`** — 个人简介

### 核心功能

- Markdown 文章渲染（含代码高亮）
- 文章列表自动从文件系统生成
- 响应式布局（手机 + 桌面）
- RSS 订阅源

### 非本期（明确不做）

- 评论系统、标签/分类、搜索、暗色模式切换、分页、管理后台

## 验收标准

1. `npm run dev` 启动开发服务器，浏览器可访问
2. `npm run build` 构建成功，输出纯静态文件
3. 新增 Markdown 文件自动出现在文章列表
4. 手机和桌面均显示正常
5. 404 页面友好提示
